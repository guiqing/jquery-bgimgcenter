jquery-bgimgcenter

jquery.bgimgcenter.js 让背景图片 无拉伸、无变形、无缝隙、很居中

产生原因

当使用背景图片的时候，使用常规的 CSS 来布局图片，都会遇到 有缝隙、被浏览器拉伸变形 等等。但是又有这种需求，我就封装了这样一个小插件。先来看下效果

插件原理

看一下源代码就可以明白，就是计算一下窗口的宽高和图片的宽高，然后计算偏移量使其居中并且占满窗口高度或者宽度，很简单的小玩意~

使用方法

引入文件

下载压缩包，将 jquery.bgimgcenter.js 文件引入页面中。或者将代码复制到项目文件中。

HTML 和 CSS 代码

为了控制更加灵活同时兼顾兼容性，所以使用 img 来“模拟”背景图片，而不是 CSS background 的方法。你需要对插件传递进去一个 img 元素。

模拟背景图片常用和推荐的结构：

<div class="body-bg">
    <img src="images/body-bg.jpg" alt="" width="1000" height="667">
</div>

通常还需要设置下面 CSS：

.body-bg{
    position: fixed;
    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
    z-index: 1;
}
.body-bg img{
    display: block;
    max-width: none;
}

这样就模拟出一个固定的背景图片。当然，上面代码只是推荐代码，你可以根据你的项目需求，自定义结构。但是需要注意的是，对于 img 不要设置 `max-width: 100%` 这样会使图片拉伸。

调用插件

使用如下代码启用这个插件：

$('.body-bg img').bgimgcenter({
    width: 1000,
    height: 667,
    target: 'window'
});

width 和 height 分别是图片的原始尺寸，是必须传递的参数。target 参数是可选参数，表示计算参照物是指当前的窗口（window）还是父元素尺寸。不设置此参数，表示根据父元素尺寸计算，具体效果如下：

依据当前窗口尺寸的效果

CSS 代码：

.body-bg{
    width: 800px;
    height: 200px;
    overflow: hidden;
}

JS 代码：

$('#target-window img').bgimgcenter({
    width: 1000,
    height: 667,
    target: 'window'
});

效果：

依据当前父级元素尺寸的效果

CSS 代码：

.body-bg{
    width: 800px;
    height: 200px;
    overflow: hidden;
}

JS 代码：

$('#target-parent img').bgimgcenter({
    width: 1000,
    height: 667
